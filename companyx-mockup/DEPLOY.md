# Company X Mockup Deployment

Suggested subdomain: `companyx.augustlyster.com`

This folder is a static website and can be deployed as-is.

Recommended Cloudflare setup:

1. Create a Cloudflare Pages project.
2. Upload the contents of this folder.
3. Add the custom domain `companyx.augustlyster.com`.
4. Cloudflare will create or suggest the needed DNS record.

Site pages:

- `index.html` - Company X consumer telco homepage
- `status.html` - lightweight mock driftstatus page
